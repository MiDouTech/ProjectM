# 静态资源（public）

此目录下的文件按原样部署，通过站点根路径访问（如 `web/public/login-bg.jpg` → `/login-bg.jpg`）。

## 登录页背景底图

登录页用 `/login-bg.jpg`（蓝色星球）作全屏底图。请把图片放到本目录并命名为 `login-bg.jpg`：

```
web/public/login-bg.jpg
```

- 缺失时登录页会优雅回落到纯 CSS 品牌渐变底图，不影响功能。
- 在 `LoginView.vue` 中以「品牌罩（暗角 + 斜向品牌渐变）+ 点阵纹理 + 柔光晕」叠加，统一在品牌氛围里又保留星球可见。

## 登录页 Logo

登录卡片顶部引用米多竖版蓝色 logo `/logo_竖_蓝色.png`。请把图片放到本目录并保持该文件名：

```
web/public/logo_竖_蓝色.png
```

- 缺失时登录页会优雅回落到「图标 + 米多」文字标记，不影响功能；放入文件后自动显示。
- 建议透明底 PNG，按 68px 视觉高度呈现、宽度自适应。
