# Triage Labels

skill 用五个标准 triage 角色描述 issue 状态机；本表把角色映射到本仓库实际使用的标签串。
本仓库无现有自定义标签，采用**默认五态**（标签串即角色名，保持英文以匹配 GitHub label）。

| 角色（mattpocock/skills） | 本仓库标签串 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需评估该 issue |
| `needs-info` | `needs-info` | 等待报告人补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整规约，AFK agent 可直接接手 |
| `ready-for-human` | `ready-for-human` | 需人工实现 |
| `wontfix` | `wontfix` | 不予处理 |

当 skill 提到某角色（如"打 AFK-ready 标签"），用本表右列对应的标签串。
若以后改用其它标签名，编辑右列即可。
